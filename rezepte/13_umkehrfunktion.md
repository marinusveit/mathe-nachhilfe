# Rezept: Umkehrfunktion

## Typische Aufgabenstellung
> „Bestimmen Sie die Umkehrfunktion f⁻¹."
> „Begründen Sie, dass f umkehrbar ist."
> „Skizzieren Sie den Graphen von f⁻¹."

## Schritt-für-Schritt: Umkehrbarkeit begründen

1. f muss **streng monoton** sein (auf dem angegebenen Intervall)
2. Nachweis: f'(x) > 0 für alle x ∈ D (streng monoton steigend)
   ODER f'(x) < 0 für alle x ∈ D (streng monoton fallend)
3. Ggf. Definitionsbereich **einschränken**, damit f monoton ist

## Schritt-für-Schritt: Umkehrfunktion berechnen

1. y = f(x) aufschreiben
2. Nach x auflösen → x = … (Ausdruck mit y)
3. x und y tauschen → f⁻¹(x) = …
4. **Definitionsbereich** von f⁻¹ = Wertebereich von f
5. **Wertebereich** von f⁻¹ = Definitionsbereich von f

## Graph
- Graph von f⁻¹ = Spiegelung des Graphen von f an der Geraden y = x

## Wichtige Umkehrfunktionen

| f(x) | f⁻¹(x) |
|---|---|
| eˣ | ln(x) |
| ln(x) | eˣ |
| x² (x ≥ 0) | √x |
| x³ | ∛x |

## Häufige Fehler
- Umkehrbarkeit behauptet, ohne Monotonie nachzuweisen
- Definitionsbereich von f⁻¹ vergessen (= Wertebereich von f!)
- x² ist NICHT umkehrbar auf ganz ℝ → Einschränkung nötig
- Beim Auflösen: ±√ vergessen und dann falsche Lösung gewählt

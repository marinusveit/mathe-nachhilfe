# Sitzung 6: Flächen zwischen Kurven & Anwendungen

## Ziel
Flächen zwischen Graphen korrekt über Schnittpunkte und Teilintervalle berechnen und Integrale in Sachkontexten sicher deuten.

## Ablauf (90 Min)

| Zeit | Was | Material |
|------|-----|----------|
| 0–20 Min | Wiederholung: Fläche zwischen zwei Kurven | Tafel + Heft |
| 20–55 Min | Notebook: Flächen, Bestandsrekonstruktion, Mittelwert | `flaechen_anwendungen.ipynb` |
| 55–80 Min | Aufgabenblatt bearbeiten | `diagnosetest.md` |
| 80–90 Min | typische Fehler im Sachkontext besprechen | — |

## Dateien

| Datei | Beschreibung |
|-------|-------------|
| `diagnosetest.md` | Aufgaben zu Flächen, Anwendungen und Mittelwert |
| `flaechen_anwendungen.ipynb` | Interaktive Anwendungen der Integralrechnung |
| `test_flaechen.py` | Fachliche Tests zu Flächen und Rekonstruktion |

## Ausführen

```bash
source ../.venv/bin/activate

# Notebook starten
jupyter lab flaechen_anwendungen.ipynb
```

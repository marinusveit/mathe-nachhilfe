# Sitzung 7: Hilfsmittelfreier Teil üben

## Ziel
Typische Teil-A-Aufgaben unter Zeitdruck trainieren und die wiederkehrenden Kurzbegründungen für das Abitur sicher beherrschen.

## Ablauf (90 Min)

| Zeit | Was | Material |
|------|-----|----------|
| 0–10 Min | Prüfungsmodus und Zeitbudget klären | — |
| 10–50 Min | Übungsklausur in Kernaufgaben bearbeiten | `diagnosetest.md` |
| 50–75 Min | Notebook: gezielte Quiz- und Begründungsaufgaben | `teil_a_training.ipynb` |
| 75–90 Min | Fehleranalyse und persönliche Checkliste | — |

## Dateien

| Datei | Beschreibung |
|-------|-------------|
| `diagnosetest.md` | Übungsklausur für den hilfsmittelfreien Teil |
| `teil_a_training.ipynb` | Interaktive Kurzformate zu Ableitungen, Stammfunktionen und Begründungen |
| `test_teil_a.py` | Fachliche Tests zur Aufgabenlogik |

## Ausführen

```bash
source ../.venv/bin/activate

# Notebook starten
jupyter lab teil_a_training.ipynb
```

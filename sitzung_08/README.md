# Sitzung 8: Probeklausur & Auswertung

## Ziel
Eine komplette Analysis-Klausur realistisch bearbeiten, systematisch auswerten und die letzten Schwachstellen gezielt nachschärfen.

## Ablauf (90 Min)

| Zeit | Was | Material |
|------|-----|----------|
| 0–55 Min | Probeklausur unter Zeitdruck bearbeiten | `probeklausur.md` |
| 55–80 Min | Musterlösung und Selbstauswertung | `probeklausur_auswertung.ipynb` |
| 80–90 Min | letzte Themen priorisieren | — |

## Dateien

| Datei | Beschreibung |
|-------|-------------|
| `probeklausur.md` | Klausurblatt mit Teil A und Teil B |
| `probeklausur_auswertung.ipynb` | Musterlösungen, Selbstauswertung und Schwächenanalyse |
| `test_probeklausur.py` | Fachliche Tests zur Klausur |

## Ausführen

```bash
source ../.venv/bin/activate

# Auswertungsnotebook starten
jupyter lab probeklausur_auswertung.ipynb
```
